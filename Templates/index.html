About Project

AI Interview Analyzer is a student hackathon project built to help college students prepare for job interviews. Using OpenAI Whisper for speech-to-text and simple AI-based scoring, the tool gives feedback on communication skills, filler words, and answer quality — so you can practice and improve before the real interview.
